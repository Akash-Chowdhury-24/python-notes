indexing is same as string
list is mutable , that is it will change if we do it 
note -> if we do this 
•	a[1:3] = [11,22]   -> this replace the a[1] and a[2] positions values with 11 and 22
•	a[1:3] = [11,22,33,44] -> this replace the a[1] and a[2] positions values with 11 and 22 and will push the remaining elements in the a[] to left and add the 33 and 44
•	a[1] = [“akash”,”bumba”]    -> this will add a new list in place of what was in index 1
•	a[2:2] = [11,22,33]   -> since its 2:2 there is nothing to delete so it will just insert 11,22,33 in index 2
build in functions 
1.	list1.append(element) -> add at the end
2.	list1.insert(index,element) -> insert and element in the index and shift others to right , it wont replace or delete
3.	list1.extend(list2) -> add the elements to the list2 into list1
4.	list1.remove(element) -> removes the first occurrence of the element. If not found it will give an error
5.	list1.pop(index) -> deletes the element in the index and returns it
6.	list1.clear()  -> empty the list but the empty list stays in the memory
7.	max(list1)
8.	min(list1)
9.	sum(list1)
10.	list1.count(element) -> count the number of the element in the list1
11.	list1.sort()
12.	 list1.reverse()
13.	list1.index(element) -> return the first index of the element
14.	del list1 -> this will delete and remove the list from the memory
15.	del list1[2] -> delete index 2
16.	del list[1:4] -> delete from index 1 to index 3

note -> del works with any data type
