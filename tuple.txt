Creation

⦁	 	Using parentheses: t = (1, 2, 3)
⦁	 	Without parentheses (comma-separated): t = 1, 2, 3
⦁	 	Single-element tuple (comma is mandatory): t = (5,)
⦁	 	Empty tuple: t = ()
⦁	 	From other iterables: t = tuple([1, 2, 3])

Indexing

⦁	 	Positive indexing: t[0], t[1]
⦁	 	Negative indexing: t[-1], t[-2]
⦁	 	Slicing: t[1:3], t[:2]


Immutability

Tuple elements cannot be changed after creation

❌ t[0] = 10 → TypeError

You can’t add/remove elements

If a tuple contains mutable objects (like lists), those inner objects can be modified



Unpacking

Basic unpacking:

a, b, c = (1, 2, 3)


Extended unpacking:

a, *b = (1, 2, 3, 4)   # a=1, b=[2,3,4]


Swapping values:

x, y = y, x
